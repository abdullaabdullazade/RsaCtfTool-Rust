## Key takeaways

- Startup overhead: Python ×51 slower cold-start (real, but trivial).
- SQUFOF: ×3.5 real compute speedup (Rust u128 fast path vs Python bignum).
- Brent 75-bit: ×3–4 real compute speedup.
- Wiener: ×770 real compute speedup (12 s → 13 ms).
- SIQS: honest gap — Rust has no implementation; Python uses yafu.
- ECM hard: honest gap — Rust ECM too few curves; Python uses GMP-ECM.
- Original benchmark `ecm`/`ecm2`/`boneh_durfee` rows were **false positives**: both tools exited 0 without finding anything because sage is not installed. Exit code ≠ success.
