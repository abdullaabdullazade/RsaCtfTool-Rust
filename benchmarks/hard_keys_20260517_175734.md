## Honest caveats

- **SIQS / QS**: Rust stubs; Python needs `yafu`. With yafu installed Python wins for large balanced semiprimes.
- **ECM hard keys**: Python needs `sage` + GMP-ECM; Rust limited to 256-bit N with only 49 curves (B1=50 000). With sage Python wins for factors > ~25 bits.
- **ECM on easy keys** (ecm_method.pub, 1029 bits): Rust rejects immediately (>256 bit). Original benchmark falsely counted both as 'ok' by checking exit code only.
- **Brent / Pollard-ρ on 128-bit N**: Python times out (>30 s); Rust succeeds. Real ×6–×17 compute speedup.
- **Wiener**: Python 12 s compute → Rust 13 ms = ×930 real speedup.
